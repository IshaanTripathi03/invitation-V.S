# Vidisha & Shubham Wedding Website

A refactored static wedding website with separated HTML, CSS, JavaScript, and assets.

## Project Structure

```text
project/
|-- index.html
|-- css/
|   |-- styles.css
|-- js/
|   |-- script.js
|-- assets/
|   |-- images/
|   |-- icons/
|   |-- fonts/
|   |-- videos/
|-- README.md
```

## Notes

- External Google Fonts and GSAP CDN links remain in `index.html`.
- Styles live in `css/styles.css`.
- Site behavior, countdown, animations, gallery, lightbox, RSVP handling, and confetti live in `js/script.js`.
- Image assets live in `assets/images/`.
